##  Tensorflow学习

### Tensorflow的安装和配置
安装anaconda  
安装pycharm   
在pycharm中建立python3.7的python运行环境，并activate   
安装tensorflow conda install tensorflow-gpu 版本为2.0.0  
（这里会自动安装cuda和cudnn依赖库）           
安装keras conda install keras     
安装sklearn conda install sklearn
